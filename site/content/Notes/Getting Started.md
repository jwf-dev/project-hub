## Welcome

This is your first note synced from Obsidian. Edit notes in your `Publish/` folder and run:

```bash
./scripts/sync_content.sh
```

## Features

- **Live Editing**: Changes appear on the website immediately
- **Wikilinks**: Use `[[Note Name]]` to link between notes
- **Status Tracking**: Projects can be idea → active → done
- **Local Only**: Everything runs on your home network

## Tips

1. Keep sensitive content out of `Publish/`
2. Use clear titles and organize with tags
3. Run sync before checking the website
4. Dark mode works automatically

Enjoy building your personal knowledge base!
